# Анализ данных билетной платформы

## 📋 Описание проекта
Исследование поведения пользователей билетной платформы, анализ факторов, влияющих на повторные покупки и формирование профилей лояльных клиентов.

## 📊 Данные
- **Период:** июнь-октябрь 2024
- **Пользователей:** 21,622
- **Заказов:** 281,679
