# Анализ данных билетной платформы

## 📋 Описание проекта
Исследование поведения пользователей билетной платформы, анализ факторов, влияющих на повторные покупки и формирование профилей лояльных клиентов.

## 📊 Данные
- **Период:** июнь-октябрь 2024
- **Пользователей:** 21,622
- **Заказов:** 281,679

## 📈 Основные результаты

1. Профиль пользователя

- 61.5% пользователей возвращаются (2+ заказа)

- 28.8% совершают 5+ заказов (сверхлояльные)

- Типичный заказ: 3 билета, средний чек ~500 руб.

2. Точки входа (первые заказы)

- 44% — концерты

- 83% — мобильные устройства

- 33% — Каменевский регион

- 24% — оператор «Билеты без проблем»

3. Ключевые факторы возврата
|Признак|Влияние|
|-------|-------|
|Количество билетов|Пик лояльности при 2-3 билетах (73.6%)|
|Интервал между покупками|Сверхлояльные возвращаются в 2 раза быстрее|
|Регион|Разброс от 55% до 67%|
|Тип мероприятия|Выставки (64.2%) vs Спорт (55.8%)|

4. Корреляционный анализ

Наиболее связаны с количеством заказов:

- avg_tickets_count (0.225) — количество билетов

- avg_revenue_rub (0.219) — средняя выручка

- avg_days_between_orders (0.157) — интервал между покупками

У сверхлояльных пользователей (5+ заказов) формируется «золотой треугольник»:
Регион ↔ Оператор ↔ Тип мероприятия (корреляции 0.54–0.69)

5. Рекомендации

- Фокус на сегмент 2-3 билета — самая лояльная группа (73.6% возвратов)

- Укреплять связки регион + оператор + тип мероприятия

- Улучшить мобильный опыт — desktop удерживает лучше (63.8% vs 60.8%)

- Проанализировать спорт — низкий возврат (55.8%)

- Изучить сегмент 5+ билетов — аномально низкая лояльность (18.8%)
